
Калькулятор

Для компиляции запустите build.sh
Для запуска нужно запустить файл "exec"
Выражения с умноженим передавать только в кавычках, так как звездочка используется как маска в bash

Программа использует алгоритм "Сортировочная станция" для преобразования инфиксной нотации в постфиксную. 
Затем она считает.

Умеет работать со скобками. Поддерживаемые операции "+, -, *, /".
Унарный минус не распознается.
